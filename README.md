----버전4.0----
■ BMS
1. 롱노트
롱노트의 여러가지 케이스

#00313:0V0000ZZ 가장 일반적인 케이스 0V~ZZ까지 롱노트로 연결
#01214:000000ZZ ZZ이 아닌 이전 노트까지 롱노트로 연결
#01012:ZZ00000000KT0000 위와 같은 케이스
#00114:ZZ 한 마디가 롱노트
#00115:0V 롱노트 키가 아니지만 한마디를 차지하는 노트는 롱노트로

2. 구현
- 담긴 노트리스트를 정방향으로 순회하면서 각 노트이 startTick을 기록
- 역방향으로 순회하면서 롱노트키(ZZ)를 만나면 00이 아닌 다른 노트를 만날때까지 순회하다가 만나면
	-> ZZ시작지점에서 만난 노트의 startTick을 뺀다.( note.duration = zz.startTick - note.startTick )
	-> 예외가 있는데 마디에 ZZ 하나이면 그 한마디는 롱노트 하나로 기록이 된다.
	-> 정방향 때 durationTick이 결정된 경우는 그 한마디가 롱노트인 경우 밖에 없다.
