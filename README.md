----버전4.0----
■ BMS
 1. 노트 시간(SecondPerBeat) 계산이 잘 되고 있었다. 다만 변수명이 많이 헷갈리기 때문에 명확하게 리네이밍
 2. 단노트 및 BMS롱노트 값이 아닌 마디에 노트 하나 들어갔을 때는 마디 하나짜리의 롱노트가 되는데 그 부분까지 해결
 3. BMS 스크립트에서 롱노트 키값을 받아 롱노트를 구현하는 과정이 남음
    - 키값을 만나면 이전 노트들을 검사해서 00이 아닌 값을 만났을 때까지가 롱노트의 시간으로 배정된다.
    - 노트 시간 배정 방식을 이중연결리스트를 이용해 순회 및 역순회하는 방식을 이용할 예정이다.
